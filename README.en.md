## 🤔 Why this repo?

"Still asks LeetCode" is often read as a signal that a company's engineering culture is **outdated**. But in 2026, that judgment got more complicated:

- **AI makes rote memorization meaningless** — an LLM writes the optimal solution in seconds, so testing "did you memorize it" is worth less and less.
- **But algorithmic thinking still matters** — many companies use LC as a **thinking filter**, not a memory test, and allow discussion and AI assistance.
- **Formats are diverging** — some moved to real-world coding, system design, AI pairing; others still hand-write the same problems.

Rather than argue about "worth it or not," let's **lay out the facts** and let everyone decide for themselves. This repo draws no conclusions — it just provides data.

For a deeper take on the trends, see [docs/trends.md](docs/trends.md).
