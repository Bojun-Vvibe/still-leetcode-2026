# 社区行为准则

## 我们的承诺

为营造开放、友善的交流环境，我们承诺让每一位参与者都能免于骚扰，无论其经验水平、性别、性取向、残障、外貌、体型、种族、年龄、宗教或国籍。

## 行为规范

**鼓励的行为：**

- 友善、尊重不同观点和经历
- 就事论事，提供建设性反馈
- 以社区利益为重，帮助新贡献者
- 如实、客观地分享面试信息

**不可接受的行为：**

- 人身攻击、侮辱、引战
- 针对任何公司或个人的诽谤、不实指控
- 公开他人隐私信息（面试官姓名、可识别个人的细节等）
- 泄露受 NDA 保护的具体面试题
- 骚扰、歧视性言论
- 任何借本项目进行的人肉、报复或商业广告行为

## 关于公司信息的特别说明

本项目收集的是**面试形式**这类事实信息，**不是评价或攻击公司的平台**。请：

- 区分「事实」（面什么）和「评价」（好不好）——本清单只收事实。
- 数据有分歧很正常，如实并列即可，不要争论谁对谁错。
- 怀疑某条信息不实，请提 Issue / PR 理性纠错，而非指责贡献者。

## 执行

如遇违反准则的行为，请通过开 Issue（标注 `行为准则`）或私信维护者举报。维护者有权删除、编辑或拒绝不符合本准则的内容，并对反复违规者采取封禁等措施。

## 归属

本准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) v2.1，并针对本项目特点做了调整。
